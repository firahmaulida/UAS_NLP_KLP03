## 📌 Deskripsi

Aplikasi berbasis **Streamlit** untuk menghasilkan **keyphrase** (kata kunci) dari abstrak skripsi berbahasa Indonesia menggunakan model **T5-small** yang telah di-fine-tune.

Aplikasi ini dikembangkan sebagai bagian dari **Ujian Akhir Semester (UAS)** mata kuliah **Natural Language Processing (NLP)**.

---

## ✨ Fitur

| Fitur | Deskripsi |
|-------|-----------|
| 🔑 **Keyphrase Generator** | Generate keyphrase dari abstrak skripsi secara otomatis |
| 📊 **Dataset Explorer** | Eksplorasi dataset DOAJ yang digunakan untuk fine-tuning |
| 📈 **Evaluasi Model** | Perbandingan performa antara RAKE dan T5-Small |
